# Netlfix-Prediction-Engine
## required libraries                
                  
### numpy  
                  pip install numpy
### pandas  
                  pip install pandas  
### sklearn  
                  pip install sklearn  
### nltk  
                  pip install nltk
  

The project deals with finding movie and tv show recommendations for the netflix dataset by implementing the k nearest neighbour method of unsupervised machine learning.
Pass the movie name to the predict function and it will displa the 10 most similar movies.
